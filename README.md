# chirashi

Ebitengine向けパーティクルコンポーネントとエディターです。
ECSライブラリとして [donburi](https://github.com/yohamta/donburi) を利用しています。

## 特徴

- Ebitengine用のパーティクルシステム
- 編集用エディター付属
- donburi (ECS) 対応

## デモ

Web上で動作確認ができます。
